1. ## Process management commands

       * ps       - Where particular bash is running
       * top      - Explains or gives every processes
       * kill     - To remove a process
       * free     - shows the disk or tells about diskspace
       * free -h  - Explains in more detailed way
       * fuser    - Tells that where or which file system is running and attached
       * nohup    - It stores the information by making file
       * bg       – Resume a stopped job in background
       * fg       – Bring a job to foreground
       * kill PID – Stop a process by PIDcd

2. ## File system commands
   
      * ls                   - list everything that is in a directory
      * mKdir                -  To create a directory  
      * ls -l                - Explains which file is created at what time and by which user
      * pwd                  - Show current directory
      * cd<change directory> - To move from one directory to another
      * touch newfile.txt    – Create a file in Ubuntu
      * rm file              – To delete a file
      * rm -r dir            –  To delete a directory
      * zcat                 - To see the contents of zip file
      * cp                   - To copy files fom one directory to another

3. ## Networking troubleshooting
   
      * ping google.com – Check network connectivity
      * ip route        – Show routing table
      * ip addr         – Show IP address and interfaces
      * dig google.com  - Checks DNS resolution
      * curl url        – Test HTTP response
      * df -h           – Disk usage
